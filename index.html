AITY 플랫폼
Big Picture

aity.co.kr — 모듈 → 커리큘럼 → 클래스

AITY는 어떤 구조인가

📦 모듈 → 📚 커리큘럼 → 🎓 클래스 → 👥 수강생
  • 모듈: 강사가 만든 학습 단위 (슬라이드·자료·실습 1세트)
  • 커리큘럼: 모듈을 묶어 만든 회차 구성 (예: AVPN 서울시 교육청 2기)
  • 클래스: 커리큘럼을 시간표·강사·수강생과 연결한 운영 단위

이번 강의에서는 운영팀이 사전에 커리큘럼·클래스를 만들어 두고, 강사는 모듈 콘텐츠 업로드 + 수업 진행 에 집중합니다.

강사가 사용하는 4가지 핵심 기능

1. 멘토 대시보드
오늘의 클래스 · 학생 진도 · 알림
2. 수업 관리
클래스 상세 · 커리큘럼 · 출석 · 녹화본
3. Google Meet 입장
한 번 클릭으로 자동 녹화 시작
4. 게시판 / Q&A
수강생과 비동기 소통 · 댓글
💡 강사 사이트 화면 우측 상단 토글이 "강사"로 되어 있는지 항상 먼저 확인하세요.

AITY 사용법 ②
수업 운영 & 녹화

자동 녹화 → 수업 진행 → 종료 절차

입장 직후 — 녹화가 자동 활성화 되는지 확인

오른쪽 상단에 "녹화 중" 빨간 점이 표시되면 정상. Gemini 회의록 작성 권한도 함께 표시됩니다.

녹화 중 표시 확인

자동 녹화가 안 되는 경우 — 수동 녹화

더보기(⋮) → 녹화 시작 → "모든 참여자에게 실시간 스트리밍 알리기" 동의 → 녹화 시작.

⚠️ 수업 종료 절차 — 이게 가장 중요합니다

  1. 녹화를 먼저 중지 (오른쪽 상단 "녹화 중" 클릭 → 중지)
  2. "녹화 중지됨" 표시 확인
  3. 아래 빨간 통화 끊기 버튼 클릭
  4. "모든 참여자를 내보내고 통화 종료" 선택 (← 그냥 나가기 ❌)
🚫 녹화가 활성화된 상태에서는 "모든 참여자 내보내기" 버튼이 비활성화됩니다.

종료 단계 1 — 녹화 먼저 중지

종료 단계 2 — 중지 버튼 클릭

종료 단계 3 — 모든 참여자 내보내고 통화 종료

녹화본은 익일 오후 수강생 화면의 "녹화본" 탭에 자동 게시됩니다.

이번 강의의 수료 기준: 출석률 70% 이상. 강사는 시작/종료 시간을 정확히 지키는 것이 곧 수강생의 수료 가부를 결정합니다.

PART 9

운영 가이드

사전 준비 → 수업 중 → 수업 후

수업 30분 전 — 사전 준비 체크리스트

  • ✅ aity.co.kr 로그인 + 우측 상단 토글이 "강사"인지 확인
  • ✅ 멘토 대시보드 → 수업 관리 → 오늘 클래스 "상세보기"
  • ✅ 모듈 자료(슬라이드/실습) 업로드 완료 확인
  • ✅ Google Meet 입장 → 자동 녹화 확인 → 사전 음성·화면 테스트
  • ✅ 게시판에 사전 안내 글 (선택) — 늦게 올 수강생을 위한 회차 요약

수업 중 — 운영 원칙 5가지

  1. 시작·종료 시간 정확히 준수 (출석률이 곧 수료 기준)
  2. 녹화 상태 항상 확인 — 시작·중간·종료 3회 시각적 체크
  3. 채팅 + 입말 Q&A 병행 — 채팅 질문은 큰 소리로 다시 읽고 답변
  4. 실습 중간점검 30분 간격 — 막힌 수강생 1:1 음소거 해제 안내
  5. 마지막 5분 핸드오프 메모 — "다음 회차에서 무엇을 가져오는지" 발표

수업 종료 후 — 강사 액션

  • 당일 24시간 내: 게시판 미답변 질문 코멘트
  • D+1 오후: 녹화본 자동 게시 확인 (수강생이 볼 수 있는 상태)
  • 다음 회차 강사에게 핸드오프 메모 전달: "지난 회차 산출물 + 미진한 부분"
  • 운영진 단톡방에 "회차 종료" 한 줄 보고

입장 시 본인 계정 확인 — Google Meet 우측 상단

⚠️ 강사·수강생 모두 AITY 가입 이메일(=신청 시 사용한 구글 계정)로 Meet에 입장해야 합니다.
다른 계정으로 입장하면 출석 기록이 남지 않을 수 있고, 호스트(강사)에게 승인 요청 알림이 발생합니다.

Meet 입장 화면 우측 상단 이메일이 본인 계정인지 항상 확인. 다른 기기에서 통화 중이면 "Switch here" 로 디바이스 전환. 다른 계정으로 들어가 있으면 우측 상단 "Switch account" 로 본인 계정 변경 후 재입장.

비상 상황 5가지 + 즉시 대응

상황대응
녹화 자동 시작 안 됨더보기(⋮) → 녹화 시작 → "스트리밍 알리기" 동의
수강생 입장 시 승인 요청 알림수강생 신청 구글계정 ≠ Meet 입장 계정 → 운영진 호출
화면 공유 안 됨브라우저 권한 → "이 사이트의 화면 녹화 허용" 후 새로고침
오디오 끊김·울림마이크/스피커 매칭 → 헤드셋 사용 권장 (기본)
강사 본인 입장 못 함운영진 핫라인 → 임시 운영자가 Meet 시작 → 강사 일반 게스트로 입장

자주 묻는 질문

  • Q. 수업 시간 외 수강생 메시지에 응답 의무? → 게시판 24시간 내 1회 코멘트 권장. 1:1 카톡/메일 응대는 강사 재량.
  • Q. 강의 자료를 PPT로 가져가도 되나? → 가능. AITY 모듈에는 Google 슬라이드 또는 PDF 링크 임베드 권장.
  • Q. 수강생이 결석하면? → 녹화본 익일 오후 게시. 출석률 70% 미만 시 수료 불가 — 강사가 별도 보충 의무는 없음.
  • Q. 강의 변경·연기? → 운영진과 최소 D-3 사전 협의. 수강생 공지는 운영진이 담당.
  • Q. 강사료 정산은? → 별도 계약서 기준 — 운영진이 안내.
